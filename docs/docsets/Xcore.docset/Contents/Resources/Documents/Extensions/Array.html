  • index(of:)

    Returns the first index where the specified value appears in the collection.

    After using index(of:) to find the position of a particular element in a collection, you can use it to access the element by subscripting. This example shows how you can pop one of the view controller from the UINavigationController.

    let navigationController = UINavigationController()
if let index = navigationController.viewControllers.index(of: SearchViewController.self) {
    navigationController.popToViewController(at: index, animated: true)
}
