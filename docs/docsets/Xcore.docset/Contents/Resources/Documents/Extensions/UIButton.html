class UIButton : UIControl, NSCoding
  • DefaultAppearance

    This configuration exists to allow some of the properties to be configured to match app’s appearance style. The UIAppearance protocol doesn’t work when the stored properites are set using associated object.

    For example:

    // Doesn't work:
UIButton.appearance().highlightedAnimation = .scale

// Works as expected:
UIButton.defaultAppearance.highlightedAnimation = .scale
    See more
